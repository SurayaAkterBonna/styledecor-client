StyleDecor - Smart Home & Ceremony Decoration Booking System
StyleDecor is a modern, full-stack appointment management system designed to streamline the booking process for decoration services. It connects clients with professional decorators, manages on-site service workflows, and provides secure payment processing.

🚀 Key Features
Dynamic Service Discovery: Browse, search, and filter decoration packages by category and budget.

Role-Based Access Control: Distinct dashboards for Users, Admins, and Decorators with JWT-secured routing.

Smart Scheduling: Real-time decorator availability check and on-site service management.

Seamless Payments: Integrated Stripe payment gateway with transaction tracking.

Interactive UI: Animated hero section using Framer-Motion, responsive card grids, and interactive maps via React Leaflet.

Real-time Tracking: Step-by-step project status updates from "Assigned" to "Completed."

Admin Analytics: Revenue monitoring and service demand visualization using charts.

🛠 Tech Stack
Frontend
Framework: React.js

Styling: Tailwind CSS, DaisyUI

Animation: Framer-Motion

Map Integration: React Leaflet

Icons: React Icons

State Management: TanStack Query (React Query)

Forms: React Hook Form

Backend
Runtime: Node.js, Express.js

Database: MongoDB

Authentication: Firebase Auth, JSON Web Tokens (JWT)

Payments: Stripe API

Image Hosting: ImageBB API
